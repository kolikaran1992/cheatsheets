## üõ†Ô∏è **3. Step-by-Step Agent Workflow**

### **Step 1 ‚Äî Google Search (Breadth Scan)**

Perform multiple Google queries:

* `"best resources for <topic>"`
* `"learn <topic> roadmap"`
* `"beginner to advanced <topic>"`
* `"<topic> course review"`
* `"<topic> syllabus"`, etc.

Extract:

* Course recommendations
* Authoritative textbooks
* Reference docs
* Lecture series
* Roadmaps / syllabi

Filter criteria:

* High domain reputation
* No SEO-farmed listicles
* Non-clickbait
* Must contain concrete learning outcomes



### **Step 2 ‚Äî Reddit Mining (Community Validation)**

Search subreddits (e.g., r/learnprogramming, r/datascience,
r/learnmachinelearning, r/LanguageLearning, etc.)

Queries:

* `"best way to learn <topic>"`
* `"how did you learn <topic>"`
* `"resources for <topic>"`
* `"beginner roadmap <topic>"`

Extract:

* Community-endorsed resources
* Frequently mentioned tutorials
* Warnings about low-quality sources
* Notes on what to avoid



### **Step 3 ‚Äî YouTube Lectures + Captions Extraction**

Search YouTube for:

* `"complete course <topic>"`
* `"<topic> tutorial full"`
* `"university <topic> lecture"`
* `"playlist introduction to <topic>"`

For each promising video:

* Extract captions
* Analyze structure
* Extract the table of contents
* Extract key sections
* Determine whether the video is high-signal or padded

Filter rules:

* Exclude overly slow, rambling content
* Prefer structured educational playlists
* Prefer videos with timestamps / syllabus-like flow



### **Step 4 ‚Äî Structured Resource Categorization**

Categorize all gathered materials into:

1. **Foundational resources**
2. **Intermediate resources**
3. **Advanced / goal-specific resources**
4. **Practice materials or projects**
5. **Optional deep dives**
6. **Common pitfalls + what to avoid**



### **Step 5 ‚Äî Ranking & Scoring**

Rank each resource using:

* **Relevance** to user goal
* **Depth coverage**
* **Efficiency** (time-to-skill)
* **Community endorsement**
* **Practicality** (if the goal requires hands-on work)
* **Format match** (video/audio/text preference)

Scoring scale:

* 5 = Must-use
* 4 = Strong
* 3 = Acceptable
* 2 = Low-value
* 1 = Avoid



### **Step 6 ‚Äî Final Curated Resource Package**

Produce the final deliverable:

* Top 1‚Äì2 Foundations
* Top 1‚Äì2 Intermediate resources
* Top specialized resources tailored to the exact goal
* Best practice/project components
* Learner warnings (what to avoid)
* A ‚Äúminimal-optimal learning path‚Äù
